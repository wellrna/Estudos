A tag b transforma o texto em NEGRITO, mas é apenas visual, não funciona em audiodescrição

A tag i transforma o texto em ITÁLICO, mas é apenas visual, não funciona em audiodescrição

A tag u transforma o texto em SUBLINHADO, mas é apenas visual, não funciona em audiodescrição

A tag strong transforma o texto em NEGRITO e funciona me audiodescrição com uma entonação forte

A tag em transforma o texto em ITÁLICO e funciona me audiodescrição com uma entonação média

A tag q insere " no trecho de texto entre a abertura e fechamento da tag, não sei se funciona em audiodescrição

A tag ul forma uma lista desordenada. Esta tag é utilizada em conjunto com a tag li que são os itens da lista.

    Exemplo de lista desordenada ul
  • Banana
  • Maçã
  • Laranja

A tag ol forma uma lista ordenada. Esta tag é utilizada em conjunto com a tag li que são os itens da lista.

    Exemplo de lista ordenada ol
  1. Pera
  2. Uva
  3. Melão

A tag img insere uma imagem na página. Ela atua em conjunto com os atributos src="endereço da imagem" e alt="descrição da imagem para utilização em audiodescrição". Esta tag não atua em par para fechamento, seu fechamento ocorre por uma /> após estes atributos obrigatórios.

Consulte os tipos de atributos existentes: W3 Schools Tag img

A tag video insere um vídeo na página. Ela atua em conjunto com o atributo src="endereço da imagem", também deve ser unserido o atributo controls para serem colocados os controles do vídeo e obrigatoriamente deve conter uma descrição do video antes de seu fechamento.

Consulte os tipos de atributos existentes: W3 Schools Tag video

A tag br efetua a quebra de linha.

A tags Blocks ou tags Inline. As tags do tipo Blocks ocupam a largura total do container, não sendo possivel colocar outro elemento a seu lado, Já as tags Inlline possibilitam colocar outros elementos a seu lado.

Consulte os tipos de tags existentes: W3 Schools Elementos Block ou Inline list

Toda tag pode ser identificada atráves de um atributo de elemento único id ou por um grupo em que se enquadre pelo atributo class, asssim podemos efetuar modificações especificas ao elemento ou ao grupo.

Através do comando Javascript getElementById podemos selecionar um determinado elemento que identificamos com o atributo id.

Através do comando Javascript getElementByClassName podemos selecionar um determinado grupo de elementos que identificamos com o atributo class.

Através do comando Javascript getElementByTagName podemos selecionar um determinado grupo de elementos que identificamos pela tag seja div, h1, p, ul, etc.

Através do comando Javascript addEventListener podemos monitorar um elemento com atributo id ou class e indicar uma nova ação após ocorrer a ação que monitoramos.

Consulte os tipos de atributos para monitorar existentes: W3 Schools Elementos Block ou Inline list

Através do comando Javascript removeEventListener podemos remover o monitoramento de um elemento com atributo id ou class.
