Aula 12 - Heranças e definições das Tag HTML

As tags possuem definições de formatações que podem ser visualizadas em W3 Schools.

Algumas propriedades, se não especificadas, são herdadas para os descendentes

  • cor do texto
  • fonte
  • fundo

Outras não são herdadas, pois são “resetadas” a cada nível

  • borda
  • margem

Cada navegador tem sua apresentação por padrão

Uma alternativa é sobrescrever todos os valores padrão removendo-os

Arquivo com os valores para remover os padrões

Aula 15 - Tamanho

Aplicável aos elementos tipo block. Use as propriedades height(altura) e width(largura)

  • auto - o navegador define qual será o tamanho
  • initial - Define o tamanho padrão
  • inherit - Define o mesmo dos pais
  • a tamanho é definido utilizando as unidades de medida - px, %, em, rem, etc
